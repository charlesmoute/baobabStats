# baobabStats
baobabStats est un package R en français qui automatise toute la chaîne du recensement : du nettoyage de millions de données jusqu’aux rapports officiels Word, Excel et HTML. Vous décrivez votre projet dans un simple fichier Excel, l’outil fait le reste. Un logiciel robuste, taillé pour les instituts africains et conçu pour durer.
